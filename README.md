# Seimens_EDA_Embedded_Track
## Content
12 sessions in 3 months of training covered the following topics:
* MCU Basic
* GPIO
* UART 
* I2C/ SPI
* AUTOSAR Layer
* Intro to OSEK-VDX
* RTE & VFB
* CAN & CANFD
* COM Stack
* Intro to Embedded Linux
